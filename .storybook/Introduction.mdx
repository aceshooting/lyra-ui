# Lyra UI

Free, independent, MIT-licensed [Lit](https://lit.dev) web components — an open-source alternative
to [Shoelace](https://shoelace.style/) and [Web Awesome](https://webawesome.com/). Lyra covers
accessible forms, dashboards, charts, data visualization, and Conversation & Agent UI. Every
component is self-contained and retheme-able through its own `--lyra-*` design tokens, with no
competitor theme or runtime dependency required.

> **Independent implementation.** Lyra is not affiliated with or endorsed by Shoelace or Web Awesome.
> Selected Web Awesome-compatible components retain documented public names under `lyra-` to ease
> migration; Shoelace users can follow the component map in the package README. Every implementation
> and design token is owned by Lyra.

## Install

```bash
npm install @aceshooting/lyra-ui
```

## Usage

```js
import '@aceshooting/lyra-ui/components/combobox/combobox.js';
import '@aceshooting/lyra-ui/components/combobox/option.js';
```

```html
<lyra-combobox label="Fruit" with-clear>
  <lyra-option value="a">Apple</lyra-option>
  <lyra-option value="b">Banana</lyra-option>
</lyra-combobox>
```

## Theming

Components read this library's own `--lyra-*` design tokens, with sensible hardcoded
defaults — override any `--lyra-*` custom property, at the component or an ancestor, to
retheme. See the [package README](https://github.com/aceshooting/lyra-ui/tree/main/packages/lyra-ui#theming)
for the full token list.
